### URL의 구성

아래 네가지 필드로 구성됨

**프로토콜** http, https, file 등

**호스트 주소** www.naver.com

**파일 경로** /home, /index.html

**Query parameter** ?id=1&postID=1 (검색, 필터링, 데이터교환시 사용)
