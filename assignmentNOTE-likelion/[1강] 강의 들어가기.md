> 강의를 통해 어떤 결과물을 만들 수 있을지 알아봅니다. 함께 실습 환경도 구축해 보도록 하겠습니다!

1. 강의의 목표와 진행방식
    1. 강의 목표
        - API가 무엇인지 알기
        - API 용어 알기
        - API를 사용해보기 → 서비스에 적용하는 방법

        ⇒ 정보를 취사선택 할 수 있는 지식기반 마련

    2. 진행 방식
        - 개념 → 실습 → 서비스 만들기
2. 최종 결과물

    마스크 재고를 파악하고 약국의 위치를 알리기

    - 사용하는 방법
        1. 목적지를 검색
        2. 지도를 확인 ⇒ 이미 만들어진 서비스를 활용
        3. 지도에서 목적지 (약국) 확인

    - Git & Github
